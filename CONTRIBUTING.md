# Contributing

Thanks for your interest in contributing!

## How to contribute

1. Fork the repo and create a feature branch.
2. Make changes with tests or policy checks where appropriate.
3. Run `make ci` locally.
4. Open a pull request with a clear description.

## Code style

- Terraform: use `terraform fmt`.
- Rego: keep rules small, readable, and cover with tests.

## Commit messages

Use descriptive, present-tense messages.
